Overview

PKLogger

Logging class used internally for Proximity Kit. Can be used to set the logging level and see more informative inforatmion about the internal happenings within the framework.

The default log level is set to PKLogLevelError

This also provides NSLog-like macros for the different levels:

  • PKLogEmerg(msg,...)
  • PKLogAlert(msg,...)
  • PKLogCrit(msg,...)
  • PKLogError(msg,...)
  • PKLogWarn(msg,...)
  • PKLogNotice(msg,...)
  • PKLogDebug(msg,...)
