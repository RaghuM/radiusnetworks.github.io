PKManagerDelegate Protocol Reference

Conforms to NSObject
Declared in PKManagerDelegate.h

Overview

@protocol PKManagerDelegate @discussion The base delegate for Proximity Kit callbacks.

This includes both callbacks that return the PK custom classes and will proxy all the raw CoreLocation callbacks.

proximityKit:didRangeBeacons:inRegion:

@method proximityKit:didRangeBeacons:inRegion

- (void)proximityKit:(PKManager *)manager didRangeBeacons:(NSArray *)beacons inRegion:(PKRegion *)region

Discussion

@method proximityKit:didRangeBeacons:inRegion

@discussion Invoked when a new set of beacons are available in the specified region.

Beacons is an array of PKIBeacon objects.

If beacons is empty, it may be assumed no beacons that match the specified region are nearby. Similarly if a specific beacon no longer appears in beacons, it may be assumed the beacon is no longer received by the device.
